# Mascardi Yard Management System

A premium, comprehensive Yard Management System (YMS) designed for automotive yards, showrooms, and logistics hubs. Built with a focus on high-performance aesthetics, real-time tracking, and operational efficiency.

## 🚀 Features

- **📊 All Company Reports**: A centralized dashboard featuring financial summaries (Pipeline Potential, Realized Revenue), sales funnel visualizations, and yard utilization heatmaps.
- **💼 Sales CRM**: End-to-end lead management, follow-up tracking, and conversion analytics.
- **📑 Inventory Management**: Real-time stock tracking with aging reports and vehicle status monitoring.
- **🏗️ Yard Mapping**: Visual slot management with drag-and-drop movement tracking.
- **🛠️ Workshop Control**: Service bay capacity monitoring, job card management, and throughput analytics.
- **🔑 Key Control**: Secure handover auditing and digital key tracking.
- **📋 Mobile Inspections**: Comprehensive digital checklists for vehicle intake and quality control.

## 🛠️ Technology Stack

- **Core**: Vanilla JavaScript (ES Modules)
- **Styling**: Modern Vanilla CSS with CSS Variables
- **Icons**: FontAwesome 6
- **Typography**: Google Fonts (Inter & Outfit)

## 📦 Getting Started

### Prerequisites

You need a local HTTP server to serve the ES modules correctly.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sammy001-svg/mascardi-yard-management-system.git
   ```
2. Navigate to the directory:
   ```bash
   cd mascardi-yard-management-system
   ```
3. Run a local server:
   ```bash
   npx http-server -p 3000
   ```
4. Access the app at `http://localhost:3000`.
